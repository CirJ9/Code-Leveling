# Code-Leveling
Project for DCIT 26 - 

A 2D top-down RPG where programming becomes a weapon. In **Code Leveling**, players fight enemies using code syntax as attack spells, turning programming concepts into RPG combat mechanics.

## 📌 About the Game

Code Leveling is a coding-based RPG inspired by monster-battling games, where players learn and use programming concepts to defeat enemies.

The world has been corrupted by digital entities called the **Virused**, and Coders are the only ones capable of fighting back by using Earth's code as their weapon. The player takes the role of a struggling Coder who must overcome their weaknesses and discover the power of code.

## Gameplay Concept

Players use code syntax as attacks:

* Print statements become basic attacks
* Loops become special abilities
* Conditional statements become combat strategies
* More advanced coding concepts unlock stronger abilities

Correct code execution results in successful attacks, while mistakes can give enemies an advantage.

## Features

* 2D top-down RPG gameplay
* Coding-based combat system
* Programming concepts as skills/spells
* Story-driven adventure
* Enemy encounters based on coding challenges

## 📖 Story

10 years ago, the world was digitalized, causing all life to become code. During the chaos, corrupted entities known as the Virused appeared and began infecting the world's source code.

The main character, once a weak F-rank Coder, carries the guilt of failing to save his family. After encountering a mysterious AI called ALICE, he gains the ability to wield code as a weapon and discovers the power to rewrite corrupted code.
